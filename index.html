Everything you need, nothing you do not

Dual audio capture

WASAPI loopback for desktop sound plus your microphone, mixed automatically — the same core idea as OBS.

Global hotkey

Start and stop from anywhere. Default is Ctrl+Shift+R, and you can change it from the tray menu.

Clipboard ready

When recording stops, the WAV is saved and copied so you can paste straight into Audacity or similar tools.

Lives in the tray

No main window clutter. Right-click for controls, startup toggle, and opening your recordings folder.

Install in 30 seconds

  1. Download localrecord.exe
  2. Run it — look for the icon in the system tray
  3. Press Ctrl + Shift + R to record
  4. Press again to stop, then paste or open the saved file

Recordings: %LOCALAPPDATA%\localrecord\LocalRecord\recordings\

About the SmartScreen warning

LocalRecord is a small, free, open-source project. It is not code-signed because Authenticode certificates cost hundreds of dollars per year — more than this project can justify right now.

Windows SmartScreen may say the app is "potentially harmful" the first time you run it. That is normal for unsigned software, not proof that anything is wrong. The full source code is on GitHub for anyone to inspect.

To proceed:

  1. Click More info on the blue SmartScreen dialog
  2. Click Run anyway

The app uses mic capture, global hotkeys, and clipboard access — behaviors Windows monitors closely. If you prefer, build it yourself from source.
